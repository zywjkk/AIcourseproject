## 摘要

本项目基于 PyTorch 框架和 PlantVillage 番茄叶片数据集，构建了一个完整的番茄病害自动识别系统。系统采用卷积神经网络（CNN）结合迁移学习策略，支持 10 类番茄叶片的病害分类（9 种病害 + 健康）。项目完成了 6 组对比实验，覆盖了从零训练 vs 迁移学习、不同冻结策略、不同学习率以及三种骨干网络（ResNet-18、ResNet-50、MobileNet-V3 Small）的性能对比。最佳模型 MobileNet-V3 Small 在测试集上达到 **96.6%** Top-1 准确率和 **0.965** Macro F1，参数量仅约 1.53M，单张推理速度 0.000195s。此外，项目还实现了 Grad-CAM 可解释性可视化和 Gradio Web 交互界面，满足了课程的核心要求与进阶挑战。

## 一、项目背景与目标

### 1.1 项目背景

番茄在种植过程中容易受到细菌性斑点病、早疫病、晚疫病、叶霉病、斑枯病、红蜘蛛、靶斑病、黄化曲叶病毒、花叶病毒等多种病害影响。传统病害诊断依赖植保专家的实地勘察与人工目测，存在以下瓶颈：

- **耗时费力**：大面积农田的逐株巡检效率极低
- **主观性强**：不同专家对病害的判断可能存在差异
- **难以大规模推广**：专业植保人才缺口大，农业智能化需求迫切

近年来，卷积神经网络在图像识别领域取得了突破性进展。通过端到端学习，CNN 能够从叶片图像中自动提取颜色、纹理、病斑形态等特征，实现病害类别的自动分类，为智慧农业提供了有效的技术支撑。

### 1.2 项目目标

本项目严格围绕课程大作业"选题四"的要求展开，核心目标如下：

1. 构建一个能够识别 **10 类** 番茄叶片图像的深度学习分类系统

4. 输出 Loss/Accuracy 曲线、混淆矩阵、Precision/Recall/F1-score 分类报告
5. 完成迁移学习实验：**从零训练 vs 迁移学习**、**不同冻结策略**、**不同学习率** 对比
6. （进阶）搭建 Gradio Web 交互界面，对比 **至少 3 种骨干网络** 的性能差异

## 二、核心原理与算法设计

### 2.1 卷积神经网络原理

卷积神经网络通过堆叠卷积层、激活函数和池化层，逐层提取图像中的判别特征：

1. **浅层卷积**：提取边缘、颜色渐变、局部纹理等低级视觉特征
2. **中层卷积**：提取病斑形态、叶脉周边病变区域等组合特征
3. **深层卷积**：提取类别判别性的抽象特征
4. **分类层**：将特征映射到 10 个类别的 logits，经 Softmax 转化为概率

### 2.2 Tensor Shape 变换过程

以 ResNet-18 为例（输入 `[B, 3, 224, 224]`）：

```text
Input:          [B, 3, 224, 224]
conv1+bn+relu:  [B, 64, 112, 112]
maxpool:        [B, 64, 56, 56]
layer1:         [B, 64, 56, 56]
layer2:         [B, 128, 28, 28]
layer3:         [B, 256, 14, 14]
layer4:         [B, 512, 7, 7]
global avg pool: [B, 512]
classifier:     [B, 10]
```

### 2.3 ResNet 残差结构

ResNet 通过残差连接解决了深层网络训练退化问题：

```
H(x) = F(x) + x
```

其中 `x` 为输入，`F(x)` 为卷积层学习的残差映射。这种结构使梯度可以直接通过恒等路径回传，稳定了深层网络的训练。本项目使用 ResNet-18 和 ResNet-50 对比不同深度的影响。

### 2.4 迁移学习与冻结策略

迁移学习是提升小规模农业数据集分类性能的关键技术。ImageNet 预训练模型已学习到通用的边缘、纹理和颜色特征，可迁移到番茄病害识别任务。

本项目设计了 **三种冻结策略**：

| 策略 | 预训练 | 冻结方式 | 目的 |
|------|--------|----------|------|
| From Scratch | ❌ | 不冻结 | 从零训练对照 |
| fc_only | ✅ | 冻结全部 backbone，仅训练分类层 | 验证预训练特征 + 线性分类的性能 |
| fc_then_layer4 | ✅ | 阶段一冻结全部训练分类头，阶段二解冻 layer4 联合微调 | **推荐策略**，兼顾适应性和稳定性 |

### 2.5 Grad-CAM 可解释性

Grad-CAM 利用目标类别对最后卷积层特征图的梯度，计算各通道的重要性权重，生成热力图。对于病害识别任务，Grad-CAM 帮助验证模型是否关注叶片病斑区域而非背景噪声，增强系统的可解释性。

```python
transforms.Compose([
    transforms.Resize((224, 224)),          # 统一尺寸

    transforms.RandomRotation(15),          # 随机旋转 ±15°
```

### 4.3 迁移学习效果分析

```
ResNet-18 From Scratch:  93.0%
ResNet-18 Transfer:      95.4%
提升:                    +2.4%
```

ImageNet 预训练模型的通用视觉特征（边缘、纹理、颜色）迁移到番茄叶片图像后，准确率提升了 2.4 个百分点，验证了迁移学习在小规模农业数据集上的有效性。

### 4.4 冻结策略分析

```
fc_only（完全冻结 backbone）:      89.0%
fc_then_layer4（解冻后部微调）:    95.4%
提升:                            +6.4%
```

完全冻结 backbone 只能利用 ImageNet 特征做线性分类，适应能力有限。解冻 layer4 后，模型可以学习番茄病害特有的斑点纹理和病变区域特征，性能显著提升。

### 4.5 学习率对比分析

```
阶段一 LR=1e-3（baseline）:  95.4%
阶段一 LR=3e-4（lower LR）:  95.8%
提升:                       +0.4%
```

较低学习率在微调阶段有助于保持预训练特征的稳定，减少过大参数更新带来的扰动，带来小幅提升。

### 4.6 三种骨干网络对比

| 骨干网络 | Accuracy | 参数量 | 推理速度 | 适用场景 |
|----------|:--------:|:------:|:--------:|----------|
| ResNet-18 | 95.4% | 11.18M | 0.000571s | 基础对照模型 |
| ResNet-50 | 96.2% | 23.53M | 0.002033s | 高精度但参数量大 |
| **MobileNet-V3 Small** | **96.6%** | **1.53M** | **0.000195s** | **最佳选型** |

MobileNet-V3 Small 综合表现最佳：准确率最高（96.6%）、参数量仅为 ResNet-18 的 1/7、推理速度最快。这使其最适合作为 Web 演示模型，也更符合智慧农业中边缘设备部署的需求。

## 九、结论与展望

### 9.1 项目成果总结

本项目完整实现了选题四"番茄病害识别检测系统"的全部核心要求及进阶挑战：

| 要求类别 | 具体内容 | 完成情况 |
|----------|----------|:--------:|
| **核心要求** | 完整训练脚本（DataLoader + 模型 + 训练 + 验证） | ✅ |
| | 测试集 Top-1 Accuracy ≥ 90% | ✅ 最高 96.6% |
| | 模型架构图 + Tensor Shape 变换 | ✅ |
| | Loss/Accuracy 曲线 | ✅ 含过拟合标注 |
| | 混淆矩阵 + 分类报告 | ✅ |
| | 超参数对比实验 | ✅ 6 组实验覆盖多维度对比 |
| | 固定随机种子 + 可复现设计 | ✅ |
| | 模块化代码结构 | ✅ src/ 下 7 个独立模块 |
| **进阶挑战** | Web 交互界面（FastAPI + SPA / Gradio） | ✅ |
| | 至少 3 种骨干网络对比 | ✅ ResNet-18/50, MobileNet-V3 |
| | 性能对比表格和柱状图 | ✅ |
| | Grad-CAM 热力图可解释性 | ✅ |

### 9.2 核心结论

1. **迁移学习有效**：相比从零训练（93.0%），迁移学习（95.4%）提升了 2.4 个百分点
2. **冻结策略关键**：解冻后部层微调（95.4%）远优于完全冻结（89.0%），提升了 6.4 个百分点
3. **MobileNet-V3 Small 最优**：以 1.53M 参数达到 96.6% 准确率，最佳性价比
4. **Grad-CAM 验证可信**：热力图显示模型聚焦叶片病斑区域，符合人类诊断直观

### 9.3 技术展望

- 可尝试 EfficientNet、ConvNeXt 等更先进架构进一步提升精度
- 引入 Attention 机制（如 Vision Transformer）捕捉全局依赖
- 扩展至更多作物品种和病害类型，增强系统的通用性
- 优化模型量化与剪枝，部署到移动端或边缘设备实现田间实时检测
- 结合无人机巡检图像，实现大田级别的病害监测
